# A-006 問い合わせ管理ページ E2Eテスト仕様書

## テスト対象ページ
- **ページ名**: 問い合わせ管理ページ (AdminInquiriesPage)
- **ページルート**: `/admin/inquiries`
- **権限**: 管理者のみ
- **使用レイアウト**: MainLayout（管理画面用）

## テスト環境
- **前提条件（全テストケース共通）**:
  - 管理者としてログイン済み（`admin@ec-platform.local` / `TestAdmin2025!`）
  - テストデータが投入済み（問い合わせ5件、AI使用状況データ）
  - ブラウザ: Chrome最新版
  - 画面解像度: 1920x1080

## 備考

### ダミー実装について
現在のAdminInquiriesPage.tsxはダミーデータを使用した実装です。以下の機能は実際のAPIと接続されていません:

- 問い合わせ一覧取得
- 返信送信
- AI使用状況データ取得

実際のAPI実装後、以下のテストケースを追加する必要があります:
- API接続エラー時の動作確認
- ローディング状態の表示確認
- 返信送信成功/失敗時の動作確認

### 定型文リスト
以下の3つの定型文が実装されています:

1. **配送状況確認**: 「お問い合わせありがとうございます。ご注文の配送状況につきましては、マイページの注文履歴からご確認いただけます。」
2. **返品について**: 「お問い合わせありがとうございます。返品をご希望の場合は、商品到着後7日以内にご連絡ください。」
3. **在庫確認**: 「お問い合わせありがとうございます。在庫状況につきましては、商品ページにてリアルタイムで更新しております。」

### グラフライブラリ
Rechartsライブラリを使用してグラフを実装しています:
- LineChart: 問い合わせ推移グラフ
- PieChart: AI対応内訳グラフ
